# CharterAI
The purpose of this project is to build a PowerPoint Add-In to be hosted on the Microsoft Add-In store. 

### Tech Stack
-   Frontend: TypeScript, ReactJS, NextJS
-   React Libraries: NextUI, HeroIcons 
-   Backend: NodeJS
-   Database: MongoDB
-   Payment: Stripe
-   Auth: PropelAuth (SAML & Google)
